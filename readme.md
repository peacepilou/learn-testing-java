# Tests d'intégration avec SpringBoot ☕️
Tu as plusieurs algos déjà écrits. Ton objectif va être d'en écrire les tests d'intégration.

### Exemple 1 : BankAccount
Consignes de tests :
1. Crée un compte au nom de “John” avec 150€.
2. Récupère ce compte via le nom.
3. Vérifie que le solde est bien 150€ et que le nom est “John”.

### Example 3 - Catalogue de livres (TDD)
Ton objectif est de créer un catalogue de livres. Les fonctionnalités seront :
- Créer un livre, 
- Récupérer un livre par son ID, 
- Récupérer tous les livres, 
- Lever une exception personnalisée si le livre n'est pas trouvé

À toi de créer tout ce dont tu as besoin, mais garde en tête que le TDD, c'est **d'abord écrire les tests avant d'écrire le code métier**
